# Salon Onboarding Platform

A web-based onboarding platform for salons and stylists that allows businesses to register, provide service details, and manage onboarding through a multi-step form workflow.

---

## Features

- Multi-step signup flow
- Separate onboarding for stylists and salons
- Service category selection
- Business information form
- Address and service location management
- Responsive UI

---

## Tech Stack

Frontend:
- React.js
- JavaScript
- CSS

Backend:
- Django REST Framework

Database:
- MySQL
